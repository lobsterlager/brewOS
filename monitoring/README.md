# Monitoring

Real-time dashboard and fermentation tracking.

## Dashboard

Local web UI showing:
- Live temperature curve (current vs. target)
- Active mash/boil step with progress bar
- Pump & heater status indicators
- Hop schedule with confirmation buttons
- Vision feed thumbnail

## Fermentation Monitoring

Post-brew-day tracking:
- Temperature logging (target: 12°C for lager fermentation)
- Glycol chiller status
- Gravity readings (manual entry)
- Fermentation timeline with key milestones
